# Honeypot / Pixiu

In this lesson, we will introduce the Pixiu contract and stay away from Pixiu tokens. Original tutorial by the [WTF Academy](https://www.wtf.academy/en).

> Note: In English, a "Pixiu" token is usually referred to as a "Honeypot" token. In the following sections, we will use the term "Pixiu" to refer to honeypot tokens.
